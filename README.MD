Um cronômetro simples e funcional feito em Python para rodar direto no seu terminal. Ele permite tanto a contagem progressiva (começando do zero) quanto a contagem regressiva (definindo um tempo inicial).

Dois modos: Escolha entre ver o tempo subir ou configurar um limite para o tempo descer.

Controles rápidos: Você pode pausar, continuar ou sair a qualquer momento apertando apenas uma tecla.

Interface limpa: O visor se atualiza automaticamente, mantendo o terminal organizado.

Enquanto o cronômetro está rodando, você pode usar:

P: Pausa o tempo.

C: Continua a contagem.

Q: Sai do programa.

Tecnologias:
Python 3

Biblioteca msvcrt (para capturar teclas no Windows)

Biblioteca os e time (para controle de tela e tempo)
